# Football Ticket Booking System (Database Project)

## 📌 Overview

This project is a relational database design for a Football Ticket Booking System. It demonstrates ERD modeling, relational integrity, and SQL query execution.

---

## 🧱 Database Tables

- Users
- Matches
- Bookings

---

## ⚙️ Features

- One-to-Many relationships (Users → Bookings)
- Many-to-One relationships (Bookings → Matches)
- Payment status tracking
- Match status tracking
- Advanced SQL queries (JOIN, Subquery, Aggregation)

---
